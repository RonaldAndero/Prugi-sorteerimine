# Sorteerimine

Prügi sorteerimis mäng

## Alustuseks

### Vajalikud pythoni programmi lisandid

```
pip install PyGame
või
install PyGame
```

## Kuidas mängu avada?

Lae alla failid GitHubist, Kui on PyGame allalaetud, siis on võimalik mängu alustada läbi pygameversion.py koodi käivitamist.
